# Slider Patterns

This document tracks slider guidance for the baseplate.

## Current State

- No carousel or slider components exist in the frontend.
- No slider library is part of the current implementation.
- There is no active slider reference pattern in the codebase right now.

## Guidance

If `[project-name]` needs sliders later:

1. Choose the library based on the new requirements.
2. Add a dedicated doc only after the first real slider implementation exists.
3. Define behavior, accessibility, and content constraints from the current product requirements.
