# Figma References

No active Figma file key is registered in the baseplate.

## When Figma Is Added

Record each implementation target before design-to-code work starts:

- Figma file key
- Page or component name
- Node ID
- Source URL
- Implementation route or component folder

Use the Figma MCP `get_design_context` tool for node-specific implementation context, then keep the resulting code aligned with `docs/FRONTEND-STRUCTURE.md`, `docs/COMPONENT-LIBRARY.md`, and `docs/BRANDING.md`.
